# 事件提醒器

简体中文 | [English](./README_EN.md)

> 悬浮式事件提醒器，支持跑马灯通知和定时提醒，让你的生活和工作更有条理。

[![版本](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/)
[![许可证](https://img.shields.io/badge/license-MIT-green.svg)](./LICENSE)

## 功能特性

### 核心功能

- **⏰ 定时提醒**：支持一次性、每天、每周、每月多种提醒模式
- **🎨 颜色标签**：提供 8 种颜色供事件分类，快速识别重要程度
- **📋 跑马灯通知**：滚动展示今日提醒事项，一目了然
- **🔔 双重通知**：同时弹出桌面通知和页面内通知，确保不遗漏
- **📦 数据管理**：支持 JSON 格式的数据导入导出，方便备份和迁移
- **⌨️ 快捷键支持**：`Ctrl+N` 快速添加事件，`Esc` 关闭面板或弹窗
- **🖱️ 悬浮窗可拖动**：自由调整位置，自动吸附屏幕左右边缘

### 使用场景

- 📅 会议提醒：团队站会、项目评审、客户沟通
- 🍽️ 生活提醒：午餐时间、运动健身、喝水提醒
- 💊 健康提醒：服药时间、体检日期、复诊预约
- 📚 学习提醒：课程安排、作业截止、考试复习

## 安装步骤

### 方法一：开发者模式安装（推荐）

1. 下载本仓库到本地
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的「开发者模式」开关
4. 点击「加载已解压的扩展程序」按钮
5. 选择本仓库的 `extracted` 文件夹即可完成安装

### 方法二：打包安装

1. 在扩展程序页面开启「开发者模式」
2. 点击「打包扩展程序」
3. 选择本仓库文件夹
4. 生成的 `.crx` 文件可直接拖入扩展程序页面安装

## 使用指南

### 基础操作

1. **打开面板**：安装后，浏览器右侧边缘会显示一个 🔔 铃铛按钮，点击即可打开提醒面板
2. **关闭面板**：再次点击铃铛按钮或按 `Esc` 键
3. **移动位置**：拖拽面板顶部可自由调整位置
4. **调整高度**：拖拽面板底部可调整显示高度

### 添加事件

1. 点击「添加事件」按钮或按 `Ctrl+N`
2. 填写事件标题（必填）
3. 设置提醒时间和日期
4. 选择重复模式：一次性 / 每天 / 每周 / 每月
5. 选择颜色标签
6. 点击「保存」

### 管理事件

- **编辑**：点击事件卡片上的编辑图标
- **删除**：点击事件卡片上的删除图标
- **暂停/启用**：点击事件卡片上的开关按钮
- **导出数据**：点击「导出」按钮，下载 JSON 文件
- **导入数据**：点击「导入」按钮，选择之前导出的 JSON 文件

## 技术架构

- **Manifest 版本**：V3（最新版本，获得更好的性能和安全支持）
- **通知机制**：使用 Chrome Alarms API 实现精确的定时提醒
- **数据存储**：使用 Chrome Storage API，本地安全存储
- **跨页面通信**：通过 Chrome Messaging 实现后台与内容脚本的通信

## 常见问题

### Q: 为什么没有收到通知？
A: 请确保在 Chrome 设置中允许该扩展程序发送通知。进入「设置」→「隐私和安全」→「网站设置」→「通知」，确认事件提醒器被允许发送通知。

### Q: 如何修改已设置的事件？
A: 在提醒面板中点击对应事件卡片上的编辑图标即可修改。

### Q: 数据丢失了怎么办？
A: 建议定期使用「导出」功能备份数据。需要恢复时，使用「导入」功能加载备份文件。

### Q: 扩展程序不显示怎么办？
A: 尝试在 `chrome://extensions/` 页面重新加载扩展程序，或重启浏览器。
